Detailed Description

Event that represents the arrival of a packet at either an intermediate node (router) or a final destination (host).

The packet can be a FLOW, ACK, or ROUTING packet.

void receive_packet_event::runEvent ( )
virtual

If this arrival event is at a router then we consult the routing table for the link to use for this packet's destination and use it to generate a send_packet_event down that link.

If the packet is arriving at a host then we create an ACK packet, make and queue a send_packet_event for the ACK, and queue a future send_packet_event for a duplicate ACK in case we don't see the next packet in the sequence. We also remove the flow's pending duplicate ACK send_packet_event for the packet with the preceding sequence number.
