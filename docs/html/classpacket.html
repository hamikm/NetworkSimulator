Detailed Description

Describes a packet in the simulated network, which can be one of the types in the enumerated type packetType.

Note that a real packet would have a payload but this class doesn't provide for one.

packet::packet ( packet_type  type,
netflow &  parent_flow,
int  seqnum 
)

This constructor infers the size of a packet from the given type, which must be ACK or FLOW since the source and destination are inferred from a parent flow.

Parameters
typeone of the values of the packet_type enum, but must be ROUTING for this constructor
parent_flowflow to which this packet belongs. ACK and FLOW packets belong to flows.
seqnumsequence number for this packet, where the first packet in flow is numbered 1. Pass in anything (or SEQNUM_FOR_NONFLOWS) for ACK packets; their seqnums are just set to SEQNUM_FOR_NONFLOWS.
Warning
assertion is triggered if the packet type is not ACK or FLOW.
